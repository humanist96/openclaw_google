# Plan: OpenClaw + Google Cloud 튜토리얼 웹교안

## 1. 개요

| 항목 | 내용 |
|------|------|
| Feature | OpenClaw + Google Cloud 설치/운영 튜토리얼 웹교안 |
| 작성일 | 2026-03-15 |
| 대상 | 비개발자/초보자 (구글 클라우드, OpenClaw 처음 접하는 일반인) |
| 목표 | 단계별로 따라하기만 하면 OpenClaw + Google 연동 완성 |
| 출처 | 강의 스크립트 + 참고 이미지 자료 |

## 2. 요구사항

### 2.1 핵심 요구사항
- **튜토리얼 형식**: 단계별 Step-by-Step 진행
- **복사/붙여넣기**: 모든 명령어와 코드는 클릭 한 번으로 복사 가능
- **시각적 고급감**: 프로페셔널하고 세련된 웹 디자인
- **이미지 활용**: 참고 스크린샷을 적재적소에 배치
- **일반인 대상**: 전문 용어에 대한 쉬운 설명, 직관적 UI

### 2.2 콘텐츠 구조 (강의 스크립트 기반 8단계)

| Step | 제목 | 핵심 내용 |
|------|------|----------|
| 0 | 인트로 | OpenClaw 소개, 4가지 핵심 요소, 구글 연동 이유 |
| 1 | 환경 준비 | AntiGravity(VS Code) 설정, WSL 연결, 터미널 준비 |
| 2 | OpenClaw 스킬 다운로드 | NPX로 GCP OpenClaw 스킬 설치 |
| 3 | 구글 클라우드 설정 | 프로젝트 생성, API 활성화(Gmail/캘린더/시트/독스/드라이브), OAuth 동의, 클라이언트 ID |
| 4 | 구글 결제계정 등록 | 무료 크레딧 설정, 결제 계정 연결 |
| 5 | GCP VM 인스턴스 생성 | Compute Engine, e2-small 사양, Ubuntu 이미지, 25GB 디스크 |
| 6 | gog-cli와 OpenClaw 설치 | VM SSH 접속, OpenClaw 설치, gog-cli 설치, 인증 키 복사 |
| 7 | OpenClaw 및 텔레그램 봇 설정 | 온보딩, Gemini 3 Flash 모델 선택, 텔레그램 BotFather, 퍼소나 설정 |
| 8 | 구글 서비스 활용 | 메일 관리, 일정 등록/리마인더, 뉴스 자동화, 시트/독스 연동 |

### 2.3 디자인 요구사항
- 다크/라이트 모드 지원
- 진행률 표시 (Progress Bar / Step Indicator)
- 코드 블록에 복사 버튼
- 팁/경고/정보 박스 (callout)
- 반응형 레이아웃 (모바일 대응)
- 부드러운 스크롤 애니메이션
- 사이드바 네비게이션 (현재 단계 하이라이트)

### 2.4 기술 스택
- **단일 HTML 파일**: 외부 의존성 없이 독립 실행
- **CSS**: 인라인 스타일, CSS 변수 활용
- **JavaScript**: 바닐라 JS (복사 기능, 다크모드, 스크롤 등)

## 3. 이미지 전략

참고 이미지 자료는 별도 준비가 필요합니다.

### 해결 방안
1. **이미지 플레이스홀더**: 각 단계에 이미지가 들어갈 위치를 명확히 표시
2. **수동 이미지 추가**: 스크린샷을 직접 캡처하여 교체
3. **이미지 설명 텍스트**: 스크린샷이 없어도 따라할 수 있도록 텍스트 가이드 병행

## 4. 비기능 요구사항

- 파일 크기: 단일 HTML, 500KB 이내
- 로딩 속도: 즉시 렌더링 (외부 리소스 없음)
- 접근성: 키보드 네비게이션 지원
- 브라우저 호환: Chrome, Safari, Firefox, Edge

## 5. 범위 외 사항 (Out of Scope)

- 백엔드 서버 구축
- 사용자 인증/로그인
- 데이터베이스 연동
- 다국어 지원 (한국어 전용)

## 6. 리스크

| 리스크 | 영향 | 대응 |
|--------|------|------|
| 참고 이미지 미확보 | 이미지 없는 교안 | 플레이스홀더 + 수동 교체 가이드 |
| 단일 HTML 파일 크기 초과 | 로딩 지연 | 이미지 최적화, CSS/JS 최소화 |
| 구글 클라우드 UI 변경 | 스크린샷 불일치 | 텍스트 가이드 병행 |
